Course Overview

Welcome to the Graph Theory course. This interactive website replaces the traditional slide deck, offering a modular look at graph algorithms, historical problems, and modern applications ranging from GPS navigation to Google's PageRank.

Part I: Basic Theory

01. Appetizer

The origins of graph theory, from the Seven Bridges of Königsberg to modern social network analysis.

Key Topics: Euler, Hamilton, Google PageRank, GPS Routing.
Start Module

02. Fundamentals

Understanding the building blocks: nodes, edges, degrees, and special graph types.

Key Topics: Definitions, Adjacency Matrices, Bipartite Graphs.
Start Module

03. Connectivity

How networks stay connected and how we traverse them efficiently.

Key Topics: BFS, DFS, Connected Components, Strong Connectivity.
Start Module

04. Shortest Paths

Algorithms for finding optimal routes in weighted networks.

Key Topics: Dijkstra’s Algorithm, Bellman-Ford, Acyclic Graphs.
Start Module

05. Trees

Properties of trees and algorithms for constructing minimal spanning trees.

Key Topics: Kruskal's, Prim's, Cayley's Formula.
Start Module

Part II: Advanced Topics & Applications

06. Planar Graphs

Graphs that can be drawn without crossing edges and Euler's formula.

Key Topics: Euler's Formula, Kuratowski's Theorem, Dual Graphs.
Start Module

07. Network Flows

Modeling flow through capacity-constrained networks.

Key Topics: Max-Flow Min-Cut, Ford-Fulkerson, Matching Problems.
Start Module

08. Cycles & Tours

The classic problems of visiting every edge or every city exactly once.

Key Topics: Eulerian Tours, Hamiltonian Cycles, TSP.
Start Module

09. Coloring

Map coloring problems and scheduling applications.

Key Topics: Chromatic Number, Four Color Theorem, Greedy Coloring.
Start Module

10. Complexity

Understanding the computational difficulty of graph problems.

Key Topics: P vs NP, NP-Completeness, Reducibility.
